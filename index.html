    Izin Akses Tulis Diperlukan

    Google mendeteksi upaya pembuatan file baru (createFile) namun skrip Anda belum diberikan hak akses tulis ke Google Drive.

    Panduan Perbaikan Langkah-demi-Langkah:

    1. Buka editor Google Apps Script Anda.
    2. Di menu sebelah kiri, klik Setelan (Project Settings) berbentuk roda gigi.
    3. Centang kotak "Tampilkan file manifes 'appsscript.json' di editor".
    4. Klik kembali menu Editor (ikon <>) di kiri atas.
    5. Buka file appsscript.json yang kini muncul, dan tambahkan kode ini sebelum kurung kurawal penutup terakhir: 
      "oauthScopes": [
  "https://www.googleapis.com/auth/drive",
  "https://www.googleapis.com/auth/spreadsheets"
]
    6. Simpan (Ctrl+S). Lalu buka file Kode.gs.
    7. Pilih fungsi otorisasiDrive di bagian atas dan klik Jalankan (Run).
    8. Lakukan proses otorisasi (Review Permissions -> akun -> Advanced -> Go to... unsafe -> Allow).
    9. Wajib: Klik Deploy -> Manage deployments -> Edit (ikon pencil) -> Versi: New Version -> Deploy.
